# NameVids Auto Uploader

Автоматизированный инструмент для массовой загрузки и публикации видео/фото на платформе NameVids (через API `add4.oakroot.top`). Проект включает в себя как скрипт для пакетной загрузки через консоль, так и удобную веб-панель для управления процессом загрузки в фоновом режиме.

## 🌟 Особенности

- **Автоматический парсинг ключей (API Keys)**: Скрипт самостоятельно извлекает свежие ключи доступа с сайта, используя ваши Cookie.
- **Массовая загрузка (Batch Upload)**: Автоматическая обработка всех файлов в папке `uploads`.
- **Веб-панель управления (`server.js`)**: Удобный интерфейс для старта/остановки загрузки, изменения Cookie "на лету", настройки задержки между публикациями и редактирования метаданных (заголовок, описание).
- **CLI загрузчик (`upload.js`)**: Простой консольный вариант скрипта. После загрузки перемещает загруженные файлы в папку `done`.
- **Имитация действий человека**: Встроенные задержки между действиями (загрузка, сохранение, публикация), чтобы снизить вероятность блокировок.

## 🛠 Установка

1. Убедитесь, что у вас установлен [Node.js](https://nodejs.org/).
2. Склонируйте репозиторий или скачайте файлы проекта.
3. Откройте терминал в папке проекта и установите зависимости:
   ```bash
   npm install
   ```
4. Создайте файл `.env` в корне проекта (если его нет) и добавьте в него ваши Cookie:
   ```env
   MY_COOKIE="ваши_cookie_с_сайта_namevids.me"
   ```

## 🚀 Использование

Вам нужно поместить медиафайлы (видео `.mp4`, `.avi`, `.mov`, `.mkv` или изображения), которые вы хотите загрузить, в папку `uploads`.

### 1. Использование через Веб-панель (Рекомендуется)

Запустите сервер с веб-панелью:
```bash
node server.js
```
- Перейдите в браузере по адресу: [http://localhost:3000](http://localhost:3000)
- Из панели вы сможете управлять Cookie, указывать задержку, задавать общие метаданные для файлов (заголовок и описание) и контролировать очередь загрузки.
- **Внимание**: При использовании `server.js` после успешной загрузки файлы удаляются из папки `uploads`.

### 2. Прямой запуск через терминал (CLI режим)

Запустите скрипт массовой загрузки контента:
```bash
node upload.js
```
- Скрипт поочередно обработает все файлы в папке `uploads`.
- Успешно загруженные файлы будут автоматически перемещены в папку `done`.

## ⚙️ Структура проекта

- `server.js` — Сервер экспресс для работы веб-панели.
- `upload.js` — Скрипт для работы проекта исключительно через консоль.
- `public/` — Статические файлы для веб-панели (HTML, CSS, JS).
- `uploads/` — Папка для исходных файлов на загрузку.
- `done/` — Сюда перемещаются опубликованные файлы (только при работе через `upload.js`).

## ⚠️ Важное предупреждение

Данный софт создан исключительно в ознакомительных целях. Использование автоматизации может нарушать правила пользовательского соглашения платформы. Используйте на свой страх и риск.
